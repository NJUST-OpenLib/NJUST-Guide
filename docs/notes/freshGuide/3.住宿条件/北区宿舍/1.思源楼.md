思源楼位于老图书馆西侧，北侧就是大教超（*教育超市*）  
生活较为便利。  
思源楼包括 1 - 6 舍，每舍都是 4 人间上床下桌，没有阳台，没有独立卫浴。   

其中 1、2 舍为五层平顶，3-6 舍为 4 层尖顶。  

据老学长介绍，思源楼于上世纪自办公楼改造而成，因此走廊较为宽阔，宿舍布局和动线设计也非常合理。   
宿舍面积也较大。


| 项目     |详情    |
| -------- | ------ |
| 布局     | 上床下桌 |
| 人数     | 四人间  |
| 独立卫浴 | 否     |
| 阳台     | 否     |
| 楼层     | 4/5 层   |
